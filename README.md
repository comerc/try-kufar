# try-kufar

Chrome Extension для страниц Kufar. Расширение добавляет рядом с ценами в белорусских рублях примерный эквивалент в долларах.

![](assets/result.png)

Сейчас поддерживаются:

- объявления авто на `auto.kufar.by`;
- объявления недвижимости на `re.kufar.by`;
- листинги, страницы объявлений, всплывающие блоки на карте и похожие объявления.

Курс USD/BYN берётся из официального API Национального банка Республики Беларусь:

```text
https://api.nbrb.by/exrates/rates/USD?parammode=2
```

Расширение кэширует курс на 6 часов. Если API временно недоступен, используется последний сохранённый курс.

## Установка из исходников

Клонируйте репозиторий:

```bash
git clone https://github.com/comerc/try-kufar.git
cd try-kufar
```

Откройте в Chrome страницу `chrome://extensions`, включите режим разработчика, нажмите `Загрузить распакованное расширение` и выберите папку проекта.

После этого откройте подходящую страницу на Kufar.by
